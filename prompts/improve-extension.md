# Improve Extension

Update one Harness extension safely.

## Instructions

1. State the problem in one sentence.
2. Inspect the smallest relevant files under `extensions/`, `skills/`, `fixtures/`, and `evals/`.
3. Keep edits inside allowed self-improvement paths.
4. Update fixtures or eval guidance alongside behavior changes.
5. Run `npm run validate` and `npm run evals`.
6. Prepare a PR summary with evidence, risk, and rollback guidance.
